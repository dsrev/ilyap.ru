Software

  • REAPER - A very customizable DAW which is minimal in file size (10+ MiB compressed, 110+ MiB uncompressed). It is also cheaper than competitors ($60 for individual, educational or non-commercial use).
  • Ventoy - FOSS utility for creating bootable USB flash drives. Just install Ventoy once and copy the image files (.iso, .wim, .img, etc) on a flash drive, no need to reformat later. This utility has saved me several times when Rufus didn't burn images well.
  • paint.net - A freeware raster graphics editor for M$ Windows that I still use. I used it to make several logos for myself and my friends (a popular example is Daniel Myslivets).
  • Visual Studio Code - The best source code editor. (Although JetBrains Fleet was introduced not so long ago. I'll see what happen next)
